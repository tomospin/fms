## Prerequisites
1. Install [Node.js](https://nodejs.org) and [MongoDB](https://www.mongodb.com)
```
$ brew install mongodb
```
2. Install Ionic Client
```
$ npm install -g ionic cordova
```
3. In rootfolder install all the dependencies:
```
$ npm install
```
